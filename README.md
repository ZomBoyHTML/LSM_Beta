README — Desafío Maldito Subercaseaux

Una experiencia web interactiva estilo ouija creada en HTML, CSS y JavaScript para promocionar el álbum/video de Los Malditos Subercaseaux.

Características:
- Tablero ouija interactivo
- Letras distribuidas en forma curva
- Sistema secreto de palabra: DELIRIO
- Velas que se encienden por cada letra correcta
- Efectos visuales: glow rojo
- temblor de pantalla
- ambiente oscuro
- Video oculto desbloqueable
- Responsive para celular y PC

Cómo jugar:
- Haz clic en las letras del tablero.
- Debes descubrir la palabra correcta.
- Cada letra correcta encenderá una vela.

Si fallas:
- el ritual se reinicia
- las velas se apagan

Al completar la palabra:
- la pantalla tiembla
- aparece el video secreto
- el álbum comienza automáticamente

Tecnologías usadas:
- HTML5
- CSS3
- JavaScript Vanilla
